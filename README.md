# Selfbot written in discord.py.

### How to install:
## Install requirements by:

### Linux:
```sh
$ py3 install -r requirements.txt
```

### Windows:
```bash
$ pip install -r requirements.txt
```

## Then run command:

### Linux:
```sh
$ py3 surf.py
```
### Windows:
```bash
$ python surf.py
```
